# satteri-img-group

The plugin stamps `imageCount` on each `<ImgGroup>` and `context` (`grid`/`carousel`) on each `<Img>`
at build time. The values rendered below are stamped, not authored.

	<Img src="/b.jpg" />
</ImgGroup>

	<Img src="/d.jpg" />
	<Img src="/e.jpg" />
</ImgGroup>
