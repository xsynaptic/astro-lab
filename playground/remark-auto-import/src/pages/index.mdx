# remark-auto-import demo

The `<Quotation>` component below is used **without an import statement**. The
[`@xsynaptic/remark-auto-import`](https://www.npmjs.com/package/@xsynaptic/remark-auto-import)
remark plugin injects the import automatically at build time.

<Quotation cite="Philip K. Dick">
	Reality is that which, when you stop believing in it, doesn't go away.
</Quotation>

No `import Quotation from ...` appears anywhere in this file.
