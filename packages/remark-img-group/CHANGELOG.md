# @xsynaptic/remark-img-group

## 0.1.0

### Minor Changes

- Initial release: a unified/remark plugin for MDX that validates `<ImgGroup>`/`<Img>` authoring and
  stamps each group with `context` and `imageCount` for its children. Published as a reference
  implementation; the Sätteri port lives in `@xsynaptic/satteri-img-group`.
